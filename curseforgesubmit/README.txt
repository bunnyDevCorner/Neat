================================================================================
  COBBLEMON NEAT BUNNY - CURSEFORGE SUBMISSION PACKAGE
================================================================================

This directory contains everything you need to submit the mod to CurseForge.

FILES INCLUDED:
---------------
1. Neat-1.21.1-41-FABRIC.jar
   → The mod file to upload to CurseForge

2. CURSEFORGE_DESCRIPTION.md
   → Copy this content into your CurseForge project description

3. CHANGELOG.md
   → Use this for release changelogs when uploading files

4. README.md
   → Full project documentation (reference)

5. LICENSE.txt
   → License information (MIT + CC BY-NC-SA 3.0)

6. SUBMISSION_GUIDE.md
   → Step-by-step guide for submitting to CurseForge

QUICK START:
-----------
1. Read SUBMISSION_GUIDE.md for detailed instructions
2. Copy content from CURSEFORGE_DESCRIPTION.md into project description
3. Upload Neat-1.21.1-41-FABRIC.jar as your first release
4. Use CHANGELOG.md content for the release changelog

IMPORTANT REMINDERS:
-------------------
✓ Credit original authors (VazkiiMods)
✓ Include disclaimer about being an unofficial fork
✓ Link to original Neat mod
✓ Set dependencies: Fabric API, Cloth Config
✓ Mark as Minecraft 1.21.1 compatible

Good luck with your submission!
